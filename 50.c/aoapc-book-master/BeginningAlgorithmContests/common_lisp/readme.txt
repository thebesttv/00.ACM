《算法竞赛入门经典》Common Lisp代码

刘汝佳

看过《黑客与画家》么？看过的话，相信你对Lisp还是会有点兴趣的

Lisp的方言很多，但因为笔者是实用主义，最符合我个人胃口的是Common Lisp（我用vi远多于emacs，所以Emacs LISP我平时基本不用）

不过我倒是在工作中会用Lisp——AutoLISP（因为我们会编写AutoCAD插件）

那为什么要给本书的Common Lisp代码呢？因为笔者对Lisp有一种特殊的情结。

最早的时候是对SICP中这样一句话产生了兴趣：Lisp is for building organisms

如果你也对这句话很好奇，那就绝对不要错过Lisp :)

笔者的入门书籍是PG的《Ansi Common Lisp》，时间紧迫的读者也可以看《Common LISP Hints》来走马观花式的学学 :)

和Haskell一样，因为笔者也只是个实践经验很少的初学者，所以如果出现了什么非常土鳖的代码，还请高手门不要见笑，并且不吝赐教和指正 :)
